# 国語辞典アプリ

## 概要

このアプリは、microCMS のコンテンツを検索できるシンプルな国語辞典アプリです。  
検索履歴やダークモードなど、最低限の使い勝手の機能を追加しています。  
Next.js の練習用としても活用できます。

---

## 主な機能

### キーワード検索

- 検索バーに単語を入力すると、microCMS API から記事を取得して表示します。  
- Enter キーでも検索が可能です。

### 検索履歴

- 過去に検索したワードを最大 10 件まで保存します。  
- 履歴のワードをクリックすると再検索できます。  
- 不要な履歴は「全削除」ボタンでまとめて消去できます。

### ダークモード

- ページ全体の配色を切り替えることができます。  
- ボタンでライトモードとダークモードを切り替えることが可能です。

### LocalStorage

- 検索履歴はブラウザの LocalStorage に保存されるため、ページをリロードしても保持されます。

---

## 使用技術

- Next.js 16（App Router）  
- React / TypeScript  
- microCMS API  
- CSS（globals.css）  
- LocalStorage
